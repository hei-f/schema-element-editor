# 收藏管理

收藏功能让您可以保存常用的 Schema 配置，方便快速复用。

## 功能概述

收藏管理支持：

- ⭐ **保存收藏** - 将当前编辑内容保存为收藏
- 📋 **浏览收藏** - 查看所有已保存的收藏
- ✏️ **编辑收藏** - 修改收藏的名称和内容
- 🚀 **应用收藏** - 快速加载收藏内容到编辑器

<!-- screenshot: 收藏功能的入口按钮 -->

## 添加收藏

### 方法一：从编辑器添加

1. 在编辑器中编辑好内容
2. 点击工具栏的「收藏」按钮
3. 选择「添加到收藏」
4. 输入收藏名称
5. 点击确认

<!-- screenshot: 添加收藏的弹窗 -->

### 收藏内容

添加收藏时会保存：

| 保存内容    | 说明                   |
| ----------- | ---------------------- |
| Schema 数据 | 当前编辑器中的完整内容 |
| 收藏名称    | 用户输入的名称         |
| 创建时间    | 自动记录               |

## 浏览收藏

### 打开收藏列表

1. 点击工具栏的「收藏」按钮
2. 选择「浏览收藏」

<!-- screenshot: 收藏列表弹窗 -->

### 收藏列表

列表显示所有已保存的收藏，包括：

- 收藏名称
- 创建时间
- 操作按钮（应用、编辑、删除）

### 搜索收藏

如果收藏数量较多，可以使用搜索框快速查找。

## 应用收藏

### 方法一：从收藏列表应用

1. 打开收藏列表
2. 找到目标收藏
3. 点击「应用」按钮
4. 确认应用（会覆盖当前编辑内容）

<!-- screenshot: 点击应用按钮的操作 -->

### 方法二：快速应用

1. 点击「收藏」按钮
2. 在下拉菜单中直接点击收藏名称

### 应用后的变化

- 编辑器内容被替换为收藏内容
- 自动记录一条「应用收藏」的历史版本（⭐ 标记）

## 编辑收藏

### 修改收藏名称

1. 打开收藏列表
2. 点击目标收藏的「编辑」按钮
3. 修改名称
4. 点击保存

<!-- screenshot: 编辑收藏弹窗 -->

### 更新收藏内容

1. 在编辑器中加载收藏
2. 修改内容
3. 重新添加为同名收藏（会询问是否覆盖）

## 删除收藏

### 删除单个收藏

1. 打开收藏列表
2. 点击目标收藏的「删除」按钮
3. 确认删除

### 批量删除

暂不支持批量删除，需要逐个删除。

## 配置选项

在配置页面「数据管理」→「收藏配置」中设置：

| 配置项       | 说明               | 默认值 |
| ------------ | ------------------ | ------ |
| 最大收藏数量 | 收藏列表的容量上限 | 50     |

<!-- screenshot: 收藏配置界面 -->

> 💡 达到上限后需要删除旧收藏才能添加新收藏。

## 存储说明

| 存储位置 | Chrome 扩展存储（chrome.storage） |
| -------- | --------------------------------- |
| 作用域   | 全局（所有页面共享）              |
| 生命周期 | 持久保存，直到手动删除或卸载扩展  |

## 功能开关

在「功能开关」→「功能模块」中可以开启或关闭收藏功能。

关闭后，工具栏不显示收藏按钮。

## 使用技巧

### 常用配置模板

1. 创建常用的 Schema 结构模板
2. 保存为收藏
3. 需要时快速应用，再根据实际需求修改

### 调试数据备份

1. 调试时发现有用的数据
2. 立即添加到收藏
3. 后续可以随时恢复

### 分类命名

建议使用规范的命名方式，便于查找：

- `[类型] 名称` - 如 `[模板] 空白消息`
- `页面名-功能` - 如 `首页-推荐卡片`
- `日期-描述` - 如 `1201-修复前数据`

## 故障排除

### 无法添加收藏

**可能原因**：

1. 已达到最大收藏数量
2. 收藏名称为空

**解决方案**：

1. 删除不需要的收藏
2. 调大最大收藏数量配置
3. 确保输入了收藏名称

### 收藏丢失

**可能原因**：

1. Chrome 扩展被重新安装
2. 浏览器数据被清除

**预防措施**：

- 重要收藏可以通过「导出」功能备份

### 应用收藏后内容不对

**可能原因**：

1. 收藏内容与当前页面不兼容
2. 收藏时保存了错误的数据

**解决方案**：

1. 检查收藏内容是否正确
2. 编辑或删除错误的收藏
